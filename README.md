# I'm A Rhythm Bartender.
## 게임 컨셉
#### 리듬 게임 X 칵테일
##### 리듬게임하면서 칵테일 만들기
###### 음악에 맞춰 노트를 누르고, 플레이어의 성과에 따라 예쁜 칵테일이 만들어진다.

## 개발 범위
###### 시작화면, 곡 선택 화면, 인 게임 화면, 점수 화면을 만들 것이고, 곡은 다른 컨셉으로 총 3개로 할 것이다.
